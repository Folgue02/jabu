FAQs

How do I register an author that I can use for uploading artifacts?

This can be done by manually requesting it through the api with a POST request:
(http://.../api/register-author/{author-name})

How do I download an artifact from the repository?

This can be done by manually requesting it through the api with a GET request:
(http://.../api/get/{author-name}/{artifact_id}/{version}/[jar/jaburon])
